## Project Description
This is a **Placement Optimization System** built for the DAA PBL project.  
The system helps students find the best internships and placements based on their **skills, CGPA, and company requirements**.  

**Frontend** is implemented in HTML/CSS for:  
- Student login/register  
- Company login/register  
- Adding internships and placements  
- Displaying job listings 
- "Find Best Job" button

**Backend** is implemented in Java using:  
- `Storage.java` – Stores all data from frontend and backend (students, companies, internships, placements)  
- `matching.java` – Logic for recommending best jobs/internships  
- `SkillGapAnalyzer.java` – Analyzes skill gaps for students
